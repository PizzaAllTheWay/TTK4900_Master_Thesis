\subsection{Performance}
\subsubsection{Overview}
This section evaluates the computational performance of the implemented SSS SLAM system. The analysis focuses on runtime, CPU utilization, and memory usage during operation. These metrics provide an indication of whether the system can operate in real time and how demanding the different components are in terms of computational resources.
\\ \\
All measurements were collected while running the full SSS SLAM pipeline and supporting ROS2 infrastructure. The results therefore represent the performance of the complete system rather than individual isolated algorithms.
