This chapter presents the results obtained from the complete Side Scan Sonar SLAM (SSS SLAM) pipeline developed in this thesis. The goal of this chapter is to evaluate the behavior, accuracy, and computational performance of the implemented system when applied to real datasets. The evaluation covers several components of the pipeline, including state estimation, local map generation, and the overall SLAM framework.
\\ \\
First, the data used for the experiments and evaluation is described, including how the datasets were gathered and prepared. The following sections then analyze the results produced by the different algorithmic components of the system. Particular focus is placed on trajectory estimation, map consistency, and the quality of generated sonar maps.
\\ \\
In addition to algorithmic evaluation, this chapter also examines the computational performance of the system. Runtime behavior, CPU usage, and memory consumption are analyzed in order to assess whether the pipeline is suitable for real time operation on embedded robotic platforms.
\\ \\
The full implementation of the SSS SLAM pipeline used in this thesis is available as open source software:
\\ \\
\url{https://github.com/PizzaAllTheWay/SSS_SLAM_ws} \cite{SSS_SLAM_repo}
\\ \\
This repository contains the full software framework, including the ROS2 packages, processing modules, and supporting tools used to generate the results presented in this chapter.
